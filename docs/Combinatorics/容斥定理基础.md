## 枚举子集
二进制第i位0/1表示不选/选该集合
```cpp
//bit确定加减，situation是情况总数，总共有x个集合
int bit,situation=1<<x;
for(int i=1;i<situation;++i)
{
    bit=0;
    for(int j=0;j<x;++j)
    {
        if((i>>j)&1)
        {

            ++bit;
        }
    }
    (bit%2*2-1)*
}
```
